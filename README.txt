FontDale Website Project

Files included:
- index.html
- directory.html
- events.html
- about.html
- newsletter.html
- css/style.css
- js/script.js
- assets/*.svg

How to use:
1. Open index.html in a web browser.
2. Navigate through the five pages using the menu.
3. Open newsletter.html and test the form validation.

Requirement mapping:
- Five pages: completed.
- External CSS only: completed via css/style.css.
- JavaScript used: completed via js/script.js.
- Responsive design: completed with meta viewport and media queries below 540px.
- At least five validations without regex: completed on Newsletter page.
- Organized files with subdirectories: completed.
