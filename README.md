# FontDale Website

FontDale is a responsive multi-page website created for an HCI Lab project. The website represents a fictional premium lifestyle mall that provides visitors with mall information, featured tenants, upcoming events, mall background, and newsletter registration.

## Project Overview

FontDale is designed as a modern mall website that focuses on usability, visual consistency, and clear navigation. The website helps users quickly explore mall information, browse tenants, check events, and subscribe to updates.

The project consists of five main pages:

- Home
- Directory
- Events
- About Us
- Newsletter

## Website Purpose

The main purpose of this website is to provide a simple and user-friendly digital platform for FontDale mall visitors. Through this website, users can:

- Learn general information about FontDale
- View featured tenants and their categories
- Check upcoming events
- Understand the mall concept and background
- Register for newsletter updates

## Features

### 1. Responsive Navigation

The website includes a consistent navigation bar across all pages. On smaller screens, the navigation changes into a mobile-friendly toggle menu.

### 2. Home Page

The Home page introduces FontDale as a premium lifestyle mall. It includes a hero section, mall highlights, visitor essentials, operating hours, location, and call-to-action buttons.

### 3. Directory Page

The Directory page displays featured tenants using card layouts. Each tenant card includes an image, category, tenant name, and short description.

Featured tenants include:

- Family Green
- Luna Dining
- Everyday Essentials

### 4. Events Page

The Events page presents upcoming mall activities. Each event card contains the event image, date, title, and description.

Displayed events include:

- Indoor Lobby Karting
- Trampoline Ball Pit Playground
- More Upcoming Events Coming Soon

### 5. About Us Page

The About Us page explains the background and identity of FontDale, including its premium mall concept, North Gallery, South Pavilion, and Lumen Skywalk.

### 6. Newsletter Page

The Newsletter page provides a registration form where users can enter their personal information and update preferences.

The form includes validation for:

- Full name
- Email address
- Phone number
- Topic of interest
- Update frequency

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts
